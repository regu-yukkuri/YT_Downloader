==================================================
  YT Downloader
==================================================

この度は「YT Downloader」をご利用いただきありがとうございます。
本ソフトは、AI技術によって作成されています

--------------------------------------------------
■ 本ソフトの特長
--------------------------------------------------
・1080p フルHD動画、高音質MP3保存に対応
・Opus 等の特殊形式も自動解析し、互換性を確保

--------------------------------------------------

--------------------------------------------------
■ 始める前に（重要！）
--------------------------------------------------
本ソフトを正常に動作させるために、必ず以下のファイルが
「同じフォルダ」にあることを確認してください。

1. YT Downloader.exe  (本ソフト)
2. ffmpeg.exe         (動画の結合・変換に必須)
3. config.json        (フォントなどのデータを設定するもの)

※ffmpegがないと、高画質動画や音声抽出ができません。

--------------------------------------------------
■ 注意事項
--------------------------------------------------
● 長時間動画（2時間以上など）について
   ダウンロード完了後、「完了：変換・結合処理中」という表示で
   止まったように見えることがありますが、これは AI 最適化を含む
   重い変換処理を裏で行っているためです。
   数分〜数十分かかる場合がありますので、そのままお待ちください。
